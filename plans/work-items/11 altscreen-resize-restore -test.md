## Problem / Gap

Bug `#06 altscreen-resize-mismatch` was fixed but has no dedicated regression test. The scenario: resize the terminal while the alternate screen is active, then restore the primary screen. The primary screen grid should use the post-resize dimensions, not a stale pre-resize snapshot stored in `AltScreenManager`.

Without a regression test, this bug class can re-emerge silently.
