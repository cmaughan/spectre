# 18 window-title-from-neovim -feature

## Summary

The SDL window title is set once at startup and never updated. Neovim's `titlestring`
option, modified-buffer indicator (`[+]`), and session names are never reflected. Every
other Neovim GUI (neovide, goneovim, etc.) updates the title dynamically.

Flagged by **[Claude]** (QoL #3, worst feature #2) and noted by **[GPT]** as a standard
expectation.

---

## How Neovim Communicates the Title

The `ext_linegrid` UI protocol includes a `set_title` event in the `redraw` batch. When
Neovim wants to set the window title it sends:
```
["set_title", [["<title string>"]]]
```
This is already part of the msgpack-RPC `redraw` notification stream.
