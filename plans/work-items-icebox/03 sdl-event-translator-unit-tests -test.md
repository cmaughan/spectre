## Problem

`libs/draxul-window/src/sdl_event_translator.cpp` is an important native boundary with known approximation points (notably: mouse modifiers are sampled from `SDL_GetModState()` rather than carried by the event itself). This is currently untested. Platform regressions in modifier handling, display-scale events, and key translation survive static refactors undetected.

## Notes
