# 09 input-dispatcher-modal -test

## Summary

Add a test that verifies `InputDispatcher` correctly routes keyboard events based on
whether the UI panel (`IUiPanel`) wants keyboard focus. When the panel claims focus,
keys must be consumed by the panel and not forwarded to the host. When the panel
releases focus, the same key must reach the host.
