# 20 url-detection-click -feature

## Summary

Terminal URLs are rendered as plain text with no visual distinction and no interaction.
Every modern terminal (iTerm2, Kitty, WezTerm) highlights URLs and opens them on
Ctrl+Click (macOS: Cmd+Click). This is a high-visibility QoL gap.

Flagged by **[Claude]** (QoL #7, worst feature #6).

---

## Implementation Plan

### Phase 1 — URL detection in the grid

- [ ] After `grid_line` events are processed and the grid is updated, run a post-process
  pass over dirty lines to detect URL spans. Use a simple regex or state-machine scan:
  - Match `https?://[^\s]+` across consecutive cells in the same row.
  - Account for double-width cells (skip the continuation cell).
- [ ] Store detected URL regions as a parallel `std::vector<UrlSpan>` per row, where
  each span has `{col_start, col_end, url_string}`.
- [ ] Invalidate URL spans for a row whenever that row is marked dirty.

### Phase 2 — Visual underline for URL cells

- [ ] When building the GPU cell array for rendering, check if a cell falls within a
  URL span. If so, set the URL-underline style bit in the cell's style flags.
- [ ] Add a `STYLE_FLAG_URL` constant to `decoration_constants.h` (use `13 shader-constants-unification`).
- [ ] Shader renders URL cells with a distinct underline (color or style distinct from
  Neovim's own underlines — e.g., dotted or colored).

### Phase 3 — Mouse hover and click

- [ ] In the mouse-event handler (`SdlWindow` → `App` → `InputDispatcher` / host), on
  `SDL_EVENT_MOUSE_MOTION`, check if the cell under the cursor is part of a URL span.
  If so, set the OS cursor to a pointer (`SDL_SYSTEM_CURSOR_POINTER`).
- [ ] On `SDL_EVENT_MOUSE_BUTTON_DOWN` with Ctrl (or Cmd on macOS) held, if the click
  lands on a URL span:
  - Extract the URL string.
  - Invoke the platform open command:
    - macOS: `open "<url>"`
    - Windows: `ShellExecuteW(NULL, L"open", url, ...)`
  - Do NOT forward the click to Neovim.
- [ ] Without modifier held, let the click reach Neovim normally.

### Phase 4 — Config opt-out

- [ ] Add `url_detection = true/false` to `config.toml` (default `true`).
- [ ] When `false`, skip the detection pass entirely.

### Validate

- [ ] Manually test: open a file with a URL, confirm underline, hover shows pointer cursor,
  Ctrl+Click opens the browser.
- [ ] Run `py do.py smoke`.
- [ ] Run `clang-format` on all touched files.

## Acceptance Criteria

- [ ] HTTP/HTTPS URLs in grid cells are underlined with a distinct style.
- [ ] Hovering over a URL cell changes the OS cursor to a pointer.
- [ ] Ctrl+Click (Cmd+Click on macOS) opens the URL in the default browser.
- [ ] Plain clicks without modifier are forwarded to Neovim normally.
- [ ] `url_detection = false` in config disables the feature entirely.
