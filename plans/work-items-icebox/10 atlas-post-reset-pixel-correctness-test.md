## Summary
`ligature_atlas_reset_tests.cpp` tests that atlas reset is triggered. There is no test that verifies glyphs are pixel-correctly re-rendered after a reset — i.e., same `AtlasRegion` UV coordinates and same atlas texture content.

## Context
